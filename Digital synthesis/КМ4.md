## Использование последовательных интерфейсов UART и SPI

### Интерфейс U(S)ART

1. (На оценку 3) Сформировать описание USART интерфейса (ведущий и ведомый), реализовать тест по передаче данных между ними и продемонстрировать результат моделирования.

2. (На оценку 4) Выполнить предыдущее задание с аппаратной реализацией протокола UART в виде передачи данных с компьютера на ПЛИС с выводом на сигнала флага приёма данных на светодиод. Для этого можно воспользоваться скриптом на {Python}[https://github.com/einmpei/course_vlsidesign/blob/main/Digital%20synthesis/uart/comms.py] или {Bash}[https://github.com/einmpei/course_vlsidesign/blob/main/Digital%20synthesis/uart/comms.sh] с целью открытия порта USB передачи данных на COM-порт (RS-232).

3. (На оценку 5) Рассчитать результат математического выражения, заданного ниже, введя переменную с компьютера по протоколу UART (скрипты используются те же, что и в п. 2). Результат вывести либо на семисегментный индикатор, либо на LCD-экран. Число передавать одноразрядное десятичное.

|---|---|
| Вариант | Выражение |
| 1 | x^2 + 3 |
| 2 | 6*x - 3 |
| 3 | 2*x^2 + 3 |
| 4 | x^3 +2*x^2 -2 |
